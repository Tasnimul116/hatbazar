# HATBAZAR

## Project Overview
Hatbazar is a fully functional web-based application developed as part of the *Web Technology* course final project. The system is designed to act as an online agricultural marketplace where users can seamlessly buy and sell agricultural products through a simple and user-friendly interface.

## Purpose
The primary goal of Hatbazar is to connect farmers, suppliers, and consumers on a single platform, enabling efficient agricultural transactions while reducing dependency on traditional middlemen.

## Key Features
- User-friendly interface for both customers and vendors  
- Agricultural product listing and browsing  
- Buy and sell functionality for farm products  
- Vendor and customer interaction through the platform  
- Organized product display for easy navigation  

## Technologies Used
- PHP  
- HTML  
- CSS  

## Target Users
- Farmers  
- Agricultural suppliers  
- Consumers  

## Conclusion
Hatbazar provides a practical and efficient digital solution for agricultural trading. The project demonstrates the application of core web technologies to solve real-world problems in the agricultural sector while emphasizing usability and functionality.
